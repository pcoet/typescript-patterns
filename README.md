## Developing

### Install

    npm install

### Test

    npm run test

#### Run a single test

    npx jest -t '<TestName>'

For example:

    npx jest -t 'regex'

### Build

    npm run build

This tests, compiles, and lints the source code. Run this before pushing a
change.
